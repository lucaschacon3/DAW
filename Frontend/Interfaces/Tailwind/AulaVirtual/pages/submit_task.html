Basic Algebra Exercises

Objective: This assignment aims to strengthen your understanding of basic algebraic concepts and enhance your problem-solving skills. Complete the following tasks carefully, and ensure you show all necessary steps in your solutions.

Instructions:

  1. Simplify Algebraic Expressions

    Simplify the following expressions by combining like terms:

    • 3x + 5x - 7 + 2
    • 4a - 3b + 2a + 6b
    • 8x² - 4x + 3x² - 2x
  2. Solve Linear Equations

    Solve for x in the following equations:

    • 5x + 7 = 22
    • 3(2x - 4) = 18
    • (2x + 5) / 4 = 3
  3. Factorize Simple Polynomials

    Factor the following polynomials:

    • x² + 7x + 12
    • 3x² - 15x
    • 4x² - 25
  4. Operations with Algebraic Fractions

    Perform the following operations:

    • Simplify: (3x² + 5x) / x
    • Add: (4 / x) + (3 / x²)
    • Multiply: (2x + 1) / x² × x² / (x + 1)

Submission Guidelines:

  • Show all work clearly, step-by-step, for each problem.
  • Submit your answers in a neat and readable format (typed or handwritten).
  • Make sure to double-check your solutions before submission.
